# Prompt Length Experiment — 2026-06-18

## Setup
- Seed: 17415 (fixed)
- Size: 1024x1536
- Workflow: rosie
- Scene: Rosie 雙馬尾 + 大黑框眼鏡 + 藍色細肩帶雪紡紗洋裝 + 懶骨頭 + 小套房 + 廚房煮晚餐

## Results

| 字數 | 效果評語 |
|------|----------|
| 100 | ❌ 細節不足，場景感弱 |
| 200 | ✅ 基本元素到位，場景有 |
| 400 | ✅ 最好，細節豐富且自然 |
| 500 | ✅ 很好，但開始感覺稍長 |
| 700 | ✅ 不錯，但邊緣開始不穩 |
| 1000 | ❌ 過擬合，畫面僵硬 |

## Conclusion
- **最佳區間：200–700 字**
- **最甜點：400–500 字**
- 100 字太少，模型缺乏足夠上下文
- 1000 字過量，模型「過擬合」導致畫面僵硬不自然

## Key Insight
- 字數過長時，模型試圖同時滿足過多約束，反而導致構圖僵硬
- 400-500 字能在「足夠細節」和「不過度約束」之間取得最佳平衡

## 第二輪實驗 — 2026-06-18（續）

### 實驗目的
驗證「背景物件增多」是否由總字數增加導致，還是描述內容本身的問題。

### 實驗設計
- 背景固定 100 字（小套房，廚房在煮晚餐）
- 主體描述分別擴寫為 200、400、600 字
- 固定 seed 17415，尺寸 1024x1536
- 主體在前，背景在後

### 發現

#### 🚨 重大陷阱：「手放肚子」→ 孕婦效應
- **600 字版本**：模型將 Rosie 生成為明顯孕婦
- **400 字版本**：有輕微孕婦感（肚子微凸）
- **200 字版本**：正常體型
- **原因**：描述越長，「hand resting on her stomach」這個動作被重複提及、強調越多，模型將「肚子」視為重要特徵並放大
- **教訓**：`hand on stomach` 是 AI 繪圖高危描述，會導致模型生成孕婦
- **修正方案**：改為「手臂自然垂放身側」或「手搭在懶骨頭扶手上」

#### 背景控制結論
- 當背景描述固定且使用明確否定句（`clean and uncluttered`, `minimal items`）時，主體字數增加**不會**導致背景物件變多
- 背景物件增多的根本原因是**背景描述本身不夠精確**，而非總字數

#### Prompt 結構最佳實踐
- **主體在前，背景在後**：確保模型優先處理主體
- **背景用否定句明確限制**：`simple counter with minimal items, clean and uncluttered background`
- **避免在主體描述中反覆提及同一身體部位**：會被模型放大為該部位的特徵

### 修正後結果（手位置修正版）
- 200 字：✅ 正常，背景乾淨
- 400 字：✅ 正常，背景乾淨
- 600 字：✅ 正常，背景乾淨（修正手位置修正後不再出現孕婦效應）

## 第三輪實驗 — 2026-06-18（續）：模組化字數分配框架

### 實驗目的
驗證「依場景特性動態分配字數」策略，將 Prompt 拆成三個獨立模組，根據各模組的複雜度動態分配字數比重。

### 實驗設計
- 場景：冰果室（Rosie 自拍照）
- 固定 seed 17415，尺寸 1024x1536
- 三模組分配：
  - ① 人物主體（表情、動作、情緒）：80 字
  - ② 服裝飾品：60-200 字（依服裝複雜度調整）
  - ③ 背景場景：300 字（冰果室複雜場景）

### 實驗組

#### A 組：寬鬆 T-shirt + 冰果室
- ① 人物：80 字
- ② 服裝：60 字（寬鬆 T-shirt 簡單帶過）
- ③ 背景：300 字（冰果室細節）
- 結果：✅ 冰果室氛圍到位，人物自然放鬆

#### B 組：藍色細肩帶雪紡洋裝 + 冰果室（服裝 60 字）
- ② 服裝只有 60 字
- 結果：⚠️ 服裝細節略少，雪紡材質感不足

#### C 組：藍色細肩帶雪紡洋裝 + 冰果室（服裝 200 字）
- ② 服裝拉到 200 字，加入細肩帶寬度、領口剪裁、雪紡透明度、裙襬細節、腰間緞帶、顏色層次（periwinkle + lavender 陰影）
- 結果：✅ 雪紡紗材質表現很好，Hina 確認滿意

### 核心結論：「字數就是注意力預算」

| 模組 | 分配原則 | 甜點區間 |
|------|---------|---------|
| ① 人物主體（表情、動作、情緒） | 永遠保持精簡，只講重點特徵，留白讓 model 自由發揮 | 80-150 字，超過開始僵硬 |
| ② 服裝飾品 | 依服裝複雜度動態分配：精緻服裝（雪紡、多層次）給 200-400 字；簡單服裝（寬鬆 T-shirt）給 60 字 | 依複雜度 |
| ③ 背景場景 | 依場景複雜度動態分配：複雜場景（冰果室、復古餐廳）給 300 字以上；簡潔背景（廚房、臥室）固定 80-100 字 | 依複雜度 |

### 關鍵洞察
- **不同面向有不同的甜點區間**：自然氛圍要少字，服裝精緻度要多字
- **字數重心應放在最複雜的模組**：簡單的模組少寫（留白讓 model 自由發揮），複雜的模組多寫（給 model 足夠資訊正確建構）
- **這跟寫作一樣**：好的 Prompt 不是描述一切，而是知道該省略什麼

### Prompt 設計心法
> **字數就是你的注意力預算，花在哪裡，哪裡就精緻。**
> 
> 哪個模組最複雜、最容易被 model 畫錯，就把字數預算分配給它。
> 簡單的東西少寫，複雜的東西多寫。
